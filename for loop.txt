*java program to print number from 1 to 10
public class Order
{
    public static void main(String[] args)
    {
        for(int i = 1; i <= 10; i++)
        {
            System.out.println(i);
        }
    }
}
*java program to print number 10 to 1
public class Reverse 
{

    public static void main(String[] args) 
    {
        for(int i = 10; i > 0; i--)
	{
            System.out.println("i = " + i);
      }

    }
}
